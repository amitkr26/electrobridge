# User Stories

## Guest
- Browse all opportunities without logging in
- Search/filter by category, location, specialization, deadline
- Click through to apply on the source site
- Use the VLSI Academy for free learning
- Subscribe to email alerts

## Seeker (Registered)
- Save opportunities and track application status
- Build a profile and resume
- Sync Academy progress across devices
- Connect with people and message connections
- Get notified about matching opportunities and network activity

## Researcher (Seeker sub-persona)
- Filter specifically for JRF/SRF/PhD/postdoc/fellowship roles
- Filter India vs International opportunities
- Access research-specific resources

## Provider (Company / Institution / University / Lab)
- Register an org account with separate onboarding
- Post opportunities that flow into the aggregator
- Manage an org page and see followers
- Search the talent pool of open-to-work seekers

## Admin / Moderator
- Add/verify/reject opportunities
- Manage scrape sources and see scraper health
- View last inserted rows per source to catch bad data early
- Post/curate news articles

## Student
- Access the free VLSI Academy curriculum
- Track learning progress across devices
- Practice with day-wise quizzes
- Take gated assessments to unlock next tracks
- Build a portfolio through project-based learning
